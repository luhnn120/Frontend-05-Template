# 手势库与动画的应用
 1. 为轮播图绑定手势库
 2. 通过监听pan和pan end事件来处理手动移动轮播图的逻辑
 3. 创建时间线，通过一个两秒的定时器往时间线中添加动画来完成轮播图的自动播放功能
 4. 处理手动移动时，自动播放状态切换的功能
  a. 手势事件strat时，暂停时间线。同时移除定时器
  b. panend事件中处理animation造成的偏移量
  c. 结束手势事件时，重新启动定时器，开始自动播放。处理图片位置
  d. 增加flick事件处理
